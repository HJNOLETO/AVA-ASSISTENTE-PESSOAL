# 📁 Pasta BIN - Scripts de Execução

Esta pasta contém todos os scripts de automação do AVA Assistant.

## 🚀 Uso Principal

**Clique duplo em:** `start.bat`

Escolha a opção desejada no menu.

---

## 📋 Opções Disponíveis

| Opção   | Função                                        |
| ------- | --------------------------------------------- |
| **[1]** | 🧹 Limpar ambiente (cache, processos, portas) |
| **[2]** | 🟢 Iniciar dev server (Frontend + Backend)    |
| **[3]** | 🛑 Parar todos os servidores                  |
| **[4]** | 📊 Verificar processos ativos                 |
| **[0]** | ❌ Sair                                       |

---

## 📌 Links de Acesso

Quando iniciado:

- **Frontend:** http://localhost:5173
- **Backend:** http://localhost:3000

---

## ⚠️ Notas

- Para parar o servidor: Escolha [3] ou pressione CTRL+C
- O ambiente é limpo automaticamente antes de iniciar
- Todos os processos Node anteriores são eliminados
