# Prompt Profissional para Redesign de Interface (UI/UX)

## Objetivo
Solicitar um redesign completo da interface do AVA Assistant v3.1, focando em uma experiência de usuário (UX) intuitiva e uma interface (UI) moderna, limpa e profissional.

---

## Estrutura do Prompt para IA de Design / Frontend

"Atue como um Especialista Senior em UX/UI com foco em aplicações SaaS de Inteligência Artificial. Preciso de um redesign completo para o sistema **AVA Assistant v3.1**.

### 1. Conceito Visual
- **Estilo:** Clean, Minimalista e Profissional.
- **Paleta de Cores:** Cores sóbrias (Cinza espacial, Azul marinho profundo, Branco puro) com toques de cores vibrantes apenas para ações importantes (Call-to-Action) e status (Sucesso, Alerta).
- **Tipografia:** Fontes sans-serif modernas (ex: Inter, Geist, ou Roboto) para máxima legibilidade.
- **Modos:** Suporte nativo a Light Mode e Dark Mode com transições suaves.

### 2. Layout Principal (Dashboard & Sidebar)
- **Sidebar Colapsável:** Menu lateral elegante contendo ícones claros para: Chat, Documentos, Agenda, Clientes, Post-its, Painel Admin e Configurações.
- **Área de Trabalho:** Espaço central amplo, utilizando o conceito de 'cards' com bordas levemente arredondadas e sombras suaves para separar seções.

### 3. Página de Chat e Gestão de Artefatos
- **Interface de Conversa:** Estilo semelhante ao ChatGPT/Claude, mas com identidade própria.
- **Janela Lateral de Artefatos (Open WebUI Style):** Uma área dedicada à direita para exibir documentos gerados, mapas mentais, códigos ou o board de post-its, permitindo interação sem sair do contexto do chat.
- **Painel de Automação Proativa:** Widget flutuante ou seção na sidebar mostrando o status de tarefas agendadas (ex: "Backup: Concluído", "Monitoramento de Logs: Ativo").
- **Console/Terminal Integrado:** Abaixo do chat, área de console para output de automações e compilações em tempo real.
- **Input de Mensagem:** Barra de texto flutuante ou fixa na base com suporte a:
    - Ícone de microfone para comando de voz (com animação de onda sonora dinâmica).
    - Ícone de clipe para anexar múltiplos arquivos (Drag & Drop).
    - Botão de envio minimalista com feedback de 'typing'.
- **Histórico:** Lista lateral esquerda para conversas recentes, com opção de renomear e excluir.

### 4. Funcionalidades Visuais Específicas
- **Post-its:** Implementar um board interativo estilo 'Canvas' onde as notas podem ser arrastadas livremente. Adicionar indicadores visuais de alarmes e prioridade.
- **Agenda:** Calendário interativo com drag-and-drop para compromissos, utilizando cores distintas para diferentes tipos de eventos.
- **Admin Panel:** Dashboards com gráficos limpos (usando bibliotecas como Recharts ou Tremor) para visualização de dados.

### 5. Requisitos de UX
- **Responsividade:** O layout deve se adaptar perfeitamente de desktops UltraWide até dispositivos móveis.
- **Feedback Visual:** Skeleton loaders para estados de carregamento e toasts elegantes para notificações do sistema.
- **Micro-interações:** Animações sutis ao passar o mouse (hover) em botões e ao abrir modais.

### 6. Referências de Design
- Inspirar-se no layout limpo do **Linear.app**.
- Utilizar a estética de profundidade do **Raycast**.
- Seguir os padrões de componentes do **Shadcn UI** e **Tailwind CSS**.

Por favor, forneça as especificações técnicas, sugestões de componentes e, se possível, o código React/Tailwind para os componentes principais da página de chat e do dashboard."
